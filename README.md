# **DoATracker**

A handy addon for Demonology Warlocks that tracks Hand of Gul'dan casts during Dominion of Argus — or just use it as a standalone Dominion of Argus cooldown monitor :)

## **Features**

*   Track Hand of Gul'dan casts during Dominion of Argus
*   Show demon summon count based on Hand of Gul'dan casts
*   Customizable announcements for Dominion of Argus start and end

## **Slash Command**

*   `/doat`
*   `/doat announce`
